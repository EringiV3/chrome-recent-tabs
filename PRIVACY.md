# Privacy Policy — Recent Tabs Switcher

**Last updated:** May 21, 2026

Recent Tabs Switcher ("this extension") is a Chrome browser extension that helps you switch between recently active tabs using a keyboard shortcut and an overlay UI.

## Data Collection

This extension does **not** collect, store, transmit, or sell any personal information, browsing history, or usage analytics.

## Data Processing

All data processed by this extension stays on your device:

- **Tab metadata** (tab IDs, titles, URLs, favicons) is read locally to display the switcher overlay and activate the tab you select.
- **User preferences** (such as maximum tabs to display and layout mode) are saved in `chrome.storage.local`.
- **MRU order** (most recently used tab order) is tracked in local memory and storage.

No data is sent to the developer or any third party.

## External Communication

This extension does **not** communicate with external servers or third-party services. It works entirely offline after installation.

## Permissions


| Permission | Purpose                                                 |
| ---------- | ------------------------------------------------------- |
| `tabs`     | Read open tab information and activate the selected tab |
| `storage`  | Save user preferences and MRU tab order locally         |


## Changes to This Policy

If this policy changes, the updated version will be published in the extension's GitHub repository.
